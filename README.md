# Fitness Management System – Analytics Module

## Project Overview
This project focuses on building an analytics module for a fitness management system to track user workouts, activity metrics, and performance KPIs using SQL and Python.

---

## Objective
- Store structured workout and activity data
- Analyze user performance trends
- Generate weekly and monthly fitness KPIs
- Support data-driven fitness tracking and reporting

---

## Tools Used
- MySQL
- SQL
- Python
- GitHub

---

## Key KPIs
- Weekly activity duration
- Monthly workout frequency
- User consistency rate
- Performance trends over time

---

## Features
- Relational database schema design for fitness data
- SQL queries for KPI calculation
- Python-based data processing and visualization
- Structured analytics pipeline

---

## Folder Structure
sql/ → database schema and KPI queries
python/ → analysis scripts
